ToolKit.jar
============
1.15.2 - [Download](https://github.com/czQery/ToolKit/releases)

Commands
---------------
- **/crasher**
- **/skick**
- **/troll**
  - Sneak
  - Sleep
  - Close
  - Glow
  - PickUp
  - Freeze
  - FakeOp
  - Flip

Events
---------------
- **Join message**
  - Teleports players to spawn
- **Leave message**

Permissions
---------------
- **toolkit.crasher**
- **toolkit.skick**
- **toolkit.troll**
- **toolkit.bypass**
  - Gives immunity against crash and troll commands
